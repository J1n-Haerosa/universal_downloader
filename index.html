Riwayat Download
Belum ada riwayat download.
Install Termux (Android)

Download Termux dari F-Droid (bukan Play Store). Update paket dengan pkg update

Install Dependencies

pkg install python ffmpeg
Lalu: pip install yt-dlp

Izin Storage

Jalankan termux-setup-storage untuk akses folder Download HP.

Clone & Jalankan

git clone https://github.com/username/repo
Lalu: python3 server.py

Akses via Browser

Buka http://localhost:5000 di HP atau http://<IP_HP>:5000 dari PC.

🐧 Juga bisa dijalankan di Linux/VPS tanpa Termux.

🔄 Update yt-dlp: pip install -U yt-dlp

⚠️ Pastikan ffmpeg terinstall untuk konversi audio.
